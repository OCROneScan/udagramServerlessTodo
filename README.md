#  ToDos Serverless project: Udacity 

## Technologies Used 
- Restful API (Lambda Functions, API Gateway and DynamoDb)
- Client (React)

## How to get the Application up and Running?!
### Backend
To deploy an application run the following commands:

```bash
cd backend
npm install
sls deploy -v
````
### Frontend
```bash
cd client
npm install
npm run start
```
